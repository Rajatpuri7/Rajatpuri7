## About Me

I'm an M.Tech student in **Automotive Technology** at **COEP Technological University (ARAI Academy), Pune**. My research focuses on **Electric Vehicle Noise Prediction using AI/ML**, with strong interests in **ADAS**, autonomous vehicles, and deep learning for audio/signal classification.

- Currently working on **EV Noise Prediction** using CNNs, ResNet & EfficientNet with mel spectrograms
- Exploring **ADAS systems**, sensor fusion, and autonomous driving technologies
- Freelance **AI Trainer & Data Annotation Specialist** (RLHF, multimodal annotation, prompt engineering)
- Based in **Pune, Maharashtra, India**
